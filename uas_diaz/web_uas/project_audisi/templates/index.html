Sistem Manajemen Kompetisi dan Penjurian Online

Peserta Juri Admin

Fitur Peserta

Pendaftaran Kompetisi

Pendaftar dapat mendaftar untuk kompetisi tertentu, melengkapi informasi pribadi, dan mengunggah karya atau file yang diperlukan.

Status Pendaftaran

Peserta dapat melihat status pendaftaran mereka, apakah diterima atau ditolak.

Hasil Kompetisi

Peserta dapat memantau hasil kompetisi dan melihat nilai atau komentar dari juri.

Fitur Juri

Melihat Karya Peserta

Juri dapat melihat daftar karya peserta dan memberikan penilaian dan komentar untuk setiap karya.

Penilaian Juri Lain

Juri dapat melihat penilaian dari juri lain untuk membandingkan skor dan memastikan konsistensi penilaian.

Fitur Admin Kompetisi

Pengelolaan Kompetisi

Admin dapat membuat dan mengelola kompetisi, termasuk nama, deskripsi, kriteria penilaian, dan tanggal penyelenggaraan.

Persetujuan Pendaftaran

Admin dapat menyetujui atau menolak pendaftaran peserta berdasarkan kelengkapan berkas.

Pengelolaan Akun

Admin dapat mengelola akun peserta dan juri kompetisi.

Laporan Kompetisi

Admin dapat melihat laporan akhir kompetisi, termasuk daftar pemenang dan skor masing-masing peserta.
